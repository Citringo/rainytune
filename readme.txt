#############################
### RainyTune for Windows ###
#############################

RainyTune は、MIDIファイルを読み込んで、視覚的に演奏するソフトウェアです。

再生を開始すると、空から降ってくる雨粒がピアノの鍵盤に向かって降り注ぎ、ピアノを演奏するようになっています。

## 動作環境

- Windows 7以降
- SSE2 に対応したCPU
- DirectX 9.0c 以降に対応したGPU。


## 使い方

songs フォルダに、お持ちのMIDIファイルをコピーしたあと、RainyTune.exeを起動してください。

その後は、メニューからファイルを選び、[Play]ボタンをクリックすると、演奏が開始します。
